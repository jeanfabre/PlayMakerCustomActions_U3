<div id="actionDesc">
<p>
Sets a named color value in a game object's material.</p>
</div>
<div id="paramRow">
	<div id="paramName">
Game Object</div>
	<div id="paramDesc">
The GameObject that the material is applied to.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Material Index</div>
	<div id="paramDesc">
GameObjects can have multiple materials. Specify an index to target a specific material.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Material</div>
	<div id="paramDesc">
Alternatively specify a Material instead of a GameObject and Index.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Named Color</div>
	<div id="paramDesc">
A named color parameter in the shader.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Color</div>
	<div id="paramDesc">
Set the parameter value.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Every Frame</div>
	<div id="paramDesc">
Repeat every frame. Useful if the value is animated.</div>
</div>
