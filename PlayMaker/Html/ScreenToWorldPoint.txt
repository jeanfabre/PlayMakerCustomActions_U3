<div id="actionDesc">
<p>
Transforms position from screen space into world space. NOTE: Uses the MainCamera!</p>
</div>
<div id="paramRow">
	<div id="paramName">
Screen Vector</div>
	<div id="paramDesc">
Screen position as a vector.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Screen X</div>
	<div id="paramDesc">
Screen X position in pixels or normalized. See Normalized.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Screen Y</div>
	<div id="paramDesc">
Screen X position in pixels or normalized. See Normalized.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Screen Z</div>
	<div id="paramDesc">
Distance into the screen in world units.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Normalized</div>
	<div id="paramDesc">
If true, X/Y coordinates are considered normalized (0-1), otherwise they are expected to be in pixels</div>
</div>

<div id="paramRow">
	<div id="paramName">
Store World Vector</div>
	<div id="paramDesc">
Store the world position in a vector3 variable.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Store World X</div>
	<div id="paramDesc">
Store the world X position in a float variable.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Store World Y</div>
	<div id="paramDesc">
Store the world Y position in a float variable.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Store World Z</div>
	<div id="paramDesc">
Store the world Z position in a float variable.</div>
</div>

<div id="paramRow">
	<div id="paramName">
Every Frame</div>
	<div id="paramDesc">
Repeat every frame</div>
</div>
